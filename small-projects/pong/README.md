Keys :
  Space  : Start / Stop the game.
  O      : Change the difficulty.
  P      : Adds / remove p2
            + resets score.
  Z / S  : P1 move keys (AZERTY).
  W / S  : P1 move keys (QWERTY).
  Arrows : P2 move keys.
  Escape : Quit game.
